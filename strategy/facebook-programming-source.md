# Whiskey Red Saloon Facebook Programming Source

This file turns the raw Facebook screenshot export into usable source material for agents generating hero slides, calendar content, offers, proof-of-life cards, and service-tier recommendations.

Use this file as evidence, not as a finished public calendar. Dates from historical posts are campaign examples unless explicitly refreshed. When a detail is marked `needs confirmation`, do not turn it into live website copy without asking.

## Source Standard

- Primary source folder: `raw-facebook-export/`
- Screenshot capture date in filenames: June 3, 2026.
- Extraction method: OCR pass plus visual inspection of key screenshots.
- Confidence levels:
  - `verified`: Dedicated post/flyer text is readable and specific.
  - `likely`: Supported by readable evidence, but may depend on time/date/context.
  - `needs confirmation`: Visible or suggested, but too ambiguous, stale, contradictory, or partial for public use.

## Agent Rules

- Do not invent happy hour deals, drink specials, DJ hours, cover charges, giveaways, or event end times.
- Preserve the exact meaning of offers. Example: "for ladies all night, 1/2 off drinks" is not the same as "all drinks 50% off for everyone."
- If using a historical event as a campaign pattern, remove or update the old date.
- If using a current/future dated event, confirm the year and active status before public copy.
- If a source says an event "starts at 9pm," do not rewrite it as "9pm-1am" unless that end time is in the same event source.
- "All ages welcome" is verified for Wednesday line dancing only. Do not extend it to other events.
- The current `this-week-and-next.yaml` happy hour copy is not supported by this source file.

## Campaign Taxonomy For Agents

Use this vocabulary instead of the vaguer phrase "event types":

- Weekly programming: recurring day-of-week reasons to visit.
- Offer nights: recurring or periodic drink/food discounts.
- Game-day campaigns: sports watch parties, pregame/after-game traffic, free food bars, team-ticket raffles.
- Venue-partner campaigns: Huntington Center ticket giveaways, Mud Hens/Walleye traffic, nearby downtown events.
- Holiday campaigns: Valentine's, St. Patrick's, Easter, Halloween, Thanksgiving, Christmas, New Year's Eve, Labor Day.
- League/social programming: dart leagues, Midwest Singles, private line-dance events.
- Proof-of-life features: game room, mechanical horse, real crowd photos, winner photos, bartop/menu photos.
- Local identity content: Toledo Spirits/local libations, downtown Toledo, 505 Jefferson Ave, "where country roads meet city streets."

## Service-Tier Implications

This source material supports a three-tier service model where the differentiator is update frequency and campaign freshness.

  - Monday Service Industry Night
  - Tuesday Karaoke
  - Wednesday Line Dancing
  - Thursday Ladies Night
  - Friday/Saturday DJ Night
  - Game room
  - Local libations

  - Football watch parties

  - Holiday party modules
  - Dart league signups

### Higher Update Tier

- Support frequent campaign-level updates:
